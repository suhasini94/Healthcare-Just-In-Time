The following file contains instructions for running this application:

If using MacOS, download MAMP and launch the Apache web server. Edit 
the "Preferences" for MAMP in order to add this folder as the root folder.
localhost port 8888 at http://localhost:8888/ is now ready to view the
"Healthcare, Just In Time" application. If using Windows, download XAMPP instead
and launch an Apache web server. Also, ensure this folder is the root folder. 
Search for the String "Listen" to find out the port XAMPP uses and then
launch localhost at that port number in order to view the application.
